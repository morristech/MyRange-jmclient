MyRange-jmclient
================

MyRange-jmclient is a J2ME client application for MyRange service.

MyRange system integrates custom mobile application, a mobile website and a website, and a web-based interface for the organizers of conferences/exibitions. 

MyRange allows participants to events on their computers and mobile phones:
- watch a program and set notifications on interesting events (fees, reports, presentations and performances);
- work with a list of all the registered participants - to search, to celebrate the interested people and schedule meetings;
- see who is within 10 meters (using Bluetooth);
- exchange electronic business cards and correspondence;

The exhibition runs at an interactive map and search for collections;
organizers have the ability to send out mass messages and watch the performance of active participants.
